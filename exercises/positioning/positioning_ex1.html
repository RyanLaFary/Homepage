Experimenting with Relative and Absolute Positioning

The position property identifies how an element is positioned on a page and whether or not it will appear within the regular flow of the page's elements. Absolute and Relative positioning works with the box offset properties - top, right, bottom, left - which identify where the element will be either positioned or moved from.

By default every element is position: static which means that it exists in the normal flow of the page and does accept any of the box offset properties.
