Advanced Selectors

Syntax

CSS has the ability to target HTML elements based on any one of their attributes. Instead of adding a class or ID to an element you can choose it based on the kind of attribute it has or what the value of that attribute is.

element[attr=value]
Select an element that has this attribute
element[attr^=value]
Select an element based on how the attribute begins
element[attr$=value]
Select an element based on how the atrtibute ends
element[attr*=value]
Select an element if the attribute contains this value anywhere
