Things to remember:

  • Use the box-sizing:border-box; property on your columns!
  • If you want the entire page to use border box you can use the universal selector as the first style rule:
    * {box-sizing: border-box;}
  • The width + margin of all three columns must total less than 100%!
Note: If we don't use border-box the actual width of boxes will be determined by width + padding + border + margin
